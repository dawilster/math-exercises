## Tooling built

- `tools/figs/_style.py` — shared grayscale-safe style (lightness + linestyle, not hue alone, so
  figures read on the e-ink reader).
- `tools/figs/mod0{0..5}_*.py` + `mod_interludes.py` — one reproducible generator per module. Module 5's
  actually trains a 2→8→1 net on two-moons by hand-rolled backprop (final loss 0.006) to snapshot the
  decision boundary learning.

## Bugs found and fixed along the way

- **`tools/render.py`**: pandoc resolved relative image paths against the CWD, not the lesson dir, so
  `--embed-resources` silently dropped every image from the print/e-reader build. Fixed with
  `--resource-path=<lesson dir>`.
- **`tools/make_epub.py`**: same class of bug — EPUB booklets would ship figureless. Fixed with a
  resource-path covering every input's directory.
- **Pedagogy bugs the figures exposed:**
  - Calculus 3.5: `lr=0.5` on x² isn't "bouncy", it solves in one step (factor 1−2η = 0). Switched the
    demo to `lr=0.9` (genuine shrinking bounce).
  - Calculus 3.5 bowl: a *circular* bowl gives a straight descent path, contradicting the "zig-zags down
    the valley" prose. Switched to an elongated bowl (honest zig-zag; also motivates momentum/Adam).
  - Algebra 0.2: the reflection figure used fudge scaling factors to fake the y=x mirror — dishonest.
    Replaced with the true inverse functions on equal axes.
